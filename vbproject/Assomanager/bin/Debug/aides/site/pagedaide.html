Authentification

AssoManager est sécurisé avec un système d'authentification à trois niveaux :

  • Administrateur (adm) : Accès complet à toutes les fonctionnalités
  • Trésorier (tre) : Accès à la gestion des adhérents, événements et documents
  • Membre (mem) : Accès uniquement aux documents

Gestion des Adhérents

Module complet pour gérer les membres de votre association :

  • Rôles : Définir les différents rôles dans l'association
  • Membres : Ajouter, modifier et supprimer les membres
  • Catégories : Classer les membres par catégories (Adhérent, Bénévole, Responsable)
  • Cotisations : Suivre les paiements de cotisations

Activités et Événements

Organisez et gérez les événements de votre association :

  • Événements : Créer et planifier des événements (tournois, assemblées générales, etc.)
  • Participants : Gérer les inscriptions et la présence aux événements
  • Suivi : Suivre le statut de participation (présent, absent, inscrit)
Haut

Gestion des Documents

Centralisez tous vos documents associatifs :

  • Statuts : Stockage des statuts de l'association
  • Procès-verbaux : Archivage des PV d'assemblées
  • Rapports financiers : Documents financiers
  • Plannings : Documents de planification
Haut

Finance de l'Association

Gérez les finances de votre association (accessible aux administrateurs uniquement) :

  • Dépenses : Enregistrer et suivre toutes les dépenses
  • Catégories de dépenses : Organiser les dépenses par catégories
  • Fournisseurs : Gérer les relations avec les fournisseurs
  • Justificatifs : Lier les documents justificatifs aux dépenses
Haut

Configuration de l'Association

Configurez les paramètres spécifiques à votre association :

  • Type d'association : Définir le type (culturelle, sportive, etc.)
  • Paramètres généraux : Nom, adresse, devise, langue
  • Utilisateurs : Gérer les comptes utilisateurs et leurs droits
